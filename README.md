# Qf Motion Commands

## 설명

서버사이드로 플레이어에게 움직임 값을 부여하는 모드.

/addmotion [entity] [x] [y] [z]
/dash [entity] [forward/backward/left/right] [power, 얼마나 이동할 지] [yheight, 얼마나 높이 뜰지]

/Open_Gui 벡터값 디버그 용 sidebar 오픈
/Close_Gui 현재 sidebar 창 닫기
